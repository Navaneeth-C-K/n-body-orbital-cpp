# n-body-orbital-cpp
A custom 2D N-body gravity with collision engine built in C++ 

An object-oriented 2D physics engine built entirely from scratch in C++.

##Important context
*The fully functioning C++ code is the file '3-body-code.cpp'. 
*The MATLAB scrip for the visualization of the data is also given here (!!!Important context: the matlab scirpt was not written by me..It was made using by ai as following the main particle was very important to me but too much work for an afternoon...tho i am working on understadning it!)
